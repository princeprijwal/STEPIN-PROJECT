### To build and execute:
```
make all
make run
```
### To execute in dev mode:
The colors are given in a string to setRubixCubeAtOnce function
```
make dev
```



### Documentation

```
make doc
```

### Test

```
make test
```

### Coverage

```
make coverage
```

### Dynamic analysis

```
make dynamic
```

### Static analysis

```
make static
```

### Uninstall
```
make clean
```
